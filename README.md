# OpenGL-app
This repository holds objects for an OpenGL apps.
## Setup
1. Clone repository using **HTTPS**
	```sh
	git clone https://github.com/Solobrosco/OpenGL-app.git
	```
2. Navigate to the repository folder
	```sh

	cd OpenGL-app

	```
3. Make glutapp
	```sh

	make

	```
4. Run OpenGL-app
    ```
    ./bin/glutapp
    ```
